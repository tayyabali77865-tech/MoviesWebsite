# Movies Website

A complete streaming platform built with Next.js, Prisma, and PostgreSQL.

## Features
- Video streaming with custom player
- Admin panel for content management
- Multi-language support
- Anime and Bollywood content
- User authentication with NextAuth

## Tech Stack
- Next.js 16
- Prisma ORM
- PostgreSQL (Neon)
- NextAuth.js
- TailwindCSS
